== Description ==

English:

This plugin (widget) shows only comments posted on pages or wiki pages. It can also be used to display comments posted on posts (like regular recent comment widget).
User can edit title of a widget and select number of comments to show, and select type of page to show comments from.

<a href="http://vukstankovic.com/en/2012/04/page-comments/">Plugin page</a>

Српски:

Овај додатак приказује најновије коментаре постављене на вашим страницама или вики страницама уколико имате инсталиран додатак.
Корисник може да измени наслов виџета и да изабере број коментара који ће бити приказани, као и да одабере тип страница са којих ће коментари бити приказивани

<a href="http://vukstankovic.com/2012/04/11/page-comments/">Страница додатка</a>

== Installation ==

English:

1. Upload `page-comments` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Page Comments widget to your sidebar

Српски:

1. Поставите фолдер `page-comments` у директоријум `/wp-content/plugins/`
2. Активирајте додатак кроз мени 'Додаци' у Вордпресу
3. Додајте Page Comments виџет у вашу бочну траку
